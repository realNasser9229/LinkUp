# LinkUp

A Discord-style real-time chat starter built with FastAPI and WebSockets.

## What you get
- polished single-page chat UI
- server switcher
- channel list
- live WebSocket room connection
- Render-ready Python deployment

## Local run

```bash
cd backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

# Enjoy using LinkUp!
